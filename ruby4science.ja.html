Ruby for Scientific Computing

Rubyの科学計算における応用例の紹介です。

このページは、RWiki:Ruby for Science に記載した古い情報の転載です。 最初のページ作成は2000年頃、最終更新は2010年12月15日です。 それ以降に登場した情報は掲載していません。 リンクの変更先がわかるものについては直しています。

Ruby科学計算関連総合

  • SciRuby - Since June 2005, by Ara T. Howard.
  • るびま0006 Ruby Library Report [第5回] 数値計算と可視化 - 著者： 堀之内 武

各分野での利用

Mathmatics - 数学

Algebra - 計算代数

  • Polynominal - by 児玉さん
  • Algebra - by 原さん - 多変数多項式環のグレブナ−基底の実装。
  • Ruby Mathematica（リンク切れ） - MathematicaへのRubyインタフェース。
  • 「プログラミングのための線形代数」 - 平岡和幸著

Earth Science - 地球科学・気象学

  • 電脳Ruby by 地球流体電脳倶楽部 - 多次元データの解析と可視化のための研究開発活動。
    • Ruby DCL - 描画ライブラリー DCL(Dennou Club Library)のRuby用ラッパー
    • RubyNetCDF
    • GAVE - Rubyで書かれた解析およびビューワー
    • GPhys - 多次元物理データ取り扱いライブラリー

Biology - 生物学

  • BioRuby - Bioinformatics に必要な機能や環境を、 Ruby を用いて統合的に実装することを目標にしたプロジェクト。
  • Ruby-Helix クライオ電子顕微鏡の写真から、高解像度の三次元構造を再構成するためのソフト。NArrayを用いている。

Chemistry - 化学

  • Chemruby - BioRuby みたいなことを化学でもやりたい。15年度未踏で開発された？

Physics - 物理

  • とんでる力学 - パリティ連載
    • 単行本(丸善 parity books)

Astronomy - 天文

  • The Kali Code for Dense Stellar Systems
  • ruby-sphere to calculate apparent positions of stars with a moderate accuracy

Materials Science - 材料科学

  • Virtual sample library

ツール・ライブラリ

  • RubyForge: Scientific/Engineering（リンク切れ）

アルゴリズム等

  • Ruby/GSL - GNU Scientific Library のラッパー
  • Ruby-SSL2 - SSLII（リンク切れ） のラッパー
  • Math::Statistics（リンク切れ） - 基本統計メソッドのMix-in。
  • ruby-cluster - 階層型クラスタリング計算。最長距離法・最短距離法・平均距離法。
  • ComputationGraph - Rubyグラフ処理エンジン。
  • Ruby GP（リンク切れ） - 遺伝的アルゴリズム・ライブラリ。
  • Ruby@RjpWiki（リンク切れ） - Rubyからの統計解析システムRの利用

数値配列

  • NArray - 多次元数値配列。FFTWのwrapper、Vector, Matrixの配列。
  • FortranArray - 多次元数値配列。BLAS, LAPACKへの自然なインタフェース。
  • linalg - DMatrix。LAPACKへのインタフェース。
  • NumArray（リンク切れ） - 多次元数値配列。欠損値処理。

数値クラス

  • Ruby-GMP（リンク切れ） - GMPのラッパー。可変長 整数、有理数、浮動小数点数。
  • BigFloat（リンク切れ） - 可変長浮動少数点クラスの拡張ライブラリ - Ruby 1.8 には BigDecimal として標準添付
  • Quanty - 単位つき量のクラス。元の単位は保持される。
  • NumRu::Units - 物理量を扱うクラス。UDUNITS Libraryのほとんどの機能をカバー。単位は常にMKSAに変換される。

データ解析

  • Ruby-CERNLIB（リンク切れ） - CERN Program Library のラッパー
  • ruby-root - Object-Oriented Data Analysis Framework Root のラッパー
  • RubyROOT - Root のSWIGラッパー

プロット・可視化

  • Gnuplot module for Ruby (rgplot) [project page] gnuplot への interface. Gordon James Miller 氏作. Ruby Gnuplot は以前の version.
  • Ruby libplot,libgraph,GDGraph,etc.
  • Ruby/PGPLOT
  • GD::Graph（リンク切れ）
  • RAA:gdchart（リンク切れ） - GDChartのRubyバインディング
  • Ruby DCL - 描画ライブラリー DCLのRuby用ラッパー
  • Ploticus from Ruby（リンク切れ） - Ploticusのラッパー
  • MRPlot（リンク切れ） - pure ruby plotting library
  • Tioga - plots using Ruby, PDF, and TeX

並列計算

  • MPI Ruby - 並列処理のためのメッセージ通信インターフェース MPI の Rubyバインディング。

計測・制御

  • Ruby for research - gpib.rb (for GPIB)
    • gpib-rb（リンク切れ）
  • IO Port Access（リンク切れ）
  • HornetsEye is a Ruby real-time computer vision extension.
